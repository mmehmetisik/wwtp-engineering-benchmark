Run the benchmark on models not yet in the leaderboard.

**To submit results:**
1. Run all 10 tasks on your model
2. Record pass/fail for each task
3. Open an [Issue](../../issues/new) with title: `[RESULTS] Model Name`
4. Include: Model name, version, scores, and any observations
